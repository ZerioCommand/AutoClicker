**AutoClicker** — вроде стандартное название но! Это комбайн, современный и скрытный инструмент для автоматизации рутинных задач и гейминга, написанный на C# (WPF). 

Помимо классического закликивания, программа поддерживает запись полноценных макросов, очередь координат, умную хуманизацию движений и кастомные темы оформления на лету.

---

## ✨ Главные возможности

* **🤖 Плавающий курсор (Humanizer):** Алгоритм имитирует погрешность человеческой руки, сдвигая курсор на 1-2 пикселя при каждом клике. Идеально для обхода базовых античитов.
* **🎥 Запись макросов (Record & Play):** Записывает все движения мыши, клики ЛКМ/ПКМ и их точные тайминги для зацикленного воспроизведения.
* **📍 Очередь координат (Multi-Point):** Задайте маршрут из неограниченного количества точек, и кликер будет методично проходить по ним.
* **🎯 Экранный оверлей (Crosshair):** Встроенный невидимый для захвата экрана прицел (точка, крест, круг) с настройкой цвета и размера.
* **👻 Скрытая привязка к окну:** Отправка кликов напрямую в свернутое или перекрытое окно через WinAPI (`PostMessage`), оставляя вашу физическую мышь свободной.
* **🎨 Движок кастомизации:** Поддержка загрузки пользовательских `.xaml` тем "на лету" без перезагрузки программы + управление прозрачностью окна.
* **💾 Система профилей:** Сохраняйте настройки под разные игры и задачи в профили (сохраняются скрыто в реестр).

## 🚀 Установка и запуск

Программа портативна и не требует установки.

1. Перейдите в раздел [Releases](../../releases) и скачайте последнюю версию `.exe` файла.
2. Запустите `AutoClicker.exe`.
3. *(Опционально)* Для установки пользовательских тем Скачайте  `Themes` и все там уже есть 1 готовая тема (поместите рядом с Кликером
